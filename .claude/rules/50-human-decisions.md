# Human Decision Rules

The following topics require a human decision before proceeding:

- LICENSE_CHANGE
- PATENT_DISCLOSURE
- LEGAL_SEMANTICS_CHOICE
- REPOSITORY_VISIBILITY
- DEFAULT_BRANCH_MERGE
- RELEASE_TAG
- PUBLIC_HISTORY_DELETION
- COMMERCIAL_RULEBASE_MIGRATION

When a task triggers a human decision:
1. Set status to HUMAN_DECISION_REQUIRED.
2. Record the decision topic in decisions.md.
3. Do not proceed with the task.
4. Other independent specs MAY continue.
